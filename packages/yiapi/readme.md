# yiapi（易接口）

## 目录结构说明

-   addons 第三方插件
-   apis 项目接口
-   config 项目配置
-   logs 日志目录
-   plugins 自己写的插件
-   tables 表定义
-   utils 工具
-   yiapi.js 易接口运行文件

## 字段类型

-   category 分类名称
-   category_id 分类 id
-   category_uuid 分类唯一 ID
-   category_code 分类编码
