## 运行

开发环境：

执行 `pnpm run dev:mp-weixin`

使用微信开发者工具，打开 `dist/dev/mp-weixin` 目录即可边开发边看效果

发布环境：

执行 `pnpm run build:mp-weixin`

使用微信开发者工具，打开 `dist/build/mp-weixin` 目录即发布项目
