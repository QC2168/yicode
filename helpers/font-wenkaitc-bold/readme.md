# 霞鹜文楷（只包含了 tc-bold）

本仓库为 `霞鹜文楷` 的 `tc-bold` 版本，旨在减少安装体积。

## 使用

导入css

```js
import '@yicode-helper/font-wenkaitc-bold';
```

使用字体

```css
body {
    font-family: 'WenKai-TC-bold';
}
```
